#  Calendar Microservice

HTTP-микросервис для управления событиями с поддержкой:

-   создания / обновления / удаления событий
-   получения событий (день / неделя / месяц)
-   фоновых напоминаний
-   архивирования прошедших событий
-   асинхронного логирования

------------------------------------------------------------------------

## Запуск

### 1. Клонировать репозиторий

``` bash
git clone <repo_url>
cd calendar
```

### 2. Установить зависимости

``` bash
go mod tidy
```

### 3. Запустить сервис

``` bash
go run main.go
```

По умолчанию сервер стартует на:

    http://localhost:8080

------------------------------------------------------------------------

## Конфигурация

    CALENDAR_PORT=8080

------------------------------------------------------------------------

## API

### Создать событие

``` bash
curl -X POST http://localhost:8080/create_event \
  -H "Content-Type: application/json" \
  -d '{
    "user_id": 1,
    "date": "2026-03-17",
    "title": "Meeting",
    "reminder": "2026-03-17T10:00:00"
  }'
```

------------------------------------------------------------------------

### Обновить событие

``` bash
curl -X POST http://localhost:8080/update_event \
  -H "Content-Type: application/json" \
  -d '{
    "id": 1,
    "user_id": 1,
    "date": "2026-03-18",
    "title": "Updated meeting"
  }'
```

------------------------------------------------------------------------

### Удалить событие

``` bash
curl -X POST http://localhost:8080/delete_event \
  -H "Content-Type: application/json" \
  -d '{
    "id": 1,
    "user_id": 1
  }'
```

------------------------------------------------------------------------

### События за день

``` bash
curl "http://localhost:8080/events_for_day?user_id=1&date=2026-03-17"
```

------------------------------------------------------------------------

### События за неделю

``` bash
curl "http://localhost:8080/events_for_week?user_id=1&date=2026-03-17"
```

------------------------------------------------------------------------

### События за месяц

``` bash
curl "http://localhost:8080/events_for_month?user_id=1&date=2026-03-17"
```

------------------------------------------------------------------------

### Архив событий

``` bash
curl "http://localhost:8080/archived_events?user_id=1"
```

------------------------------------------------------------------------

## Архитектура

    internal/
