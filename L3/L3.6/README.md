# SalesTracker

**SalesTracker** --- сервис для учета транзакций (доходов и расходов) с
аналитикой и экспортом данных.\
Проект демонстрирует работу с PostgreSQL, SQL-агрегациями, REST API и
простым веб‑интерфейсом.

Сервис умеет:

-   выполнять CRUD операции с транзакциями
-   строить агрегированную аналитику
-   группировать данные (по дням, неделям, категориям)
-   рассчитывать медиану и 90‑й перцентиль
-   экспортировать отчеты в CSV
-   работать через Web UI

------------------------------------------------------------------------

## Основные возможности

### Транзакции

-   Создание записи (доход / расход)
-   Просмотр списка транзакций
-   Фильтрация по:
    -   типу
    -   категории
    -   периоду
-   Обновление записи
-   Удаление записи

### Аналитика

Сервис рассчитывает:

-   сумму (sum)
-   среднее значение (avg)
-   количество (count)
-   медиану
-   90‑й перцентиль
-   баланс (доходы − расходы)

Поддерживается группировка:

-   по дням
-   по неделям
-   по категориям

### Экспорт данных

Можно скачать отчет:

CSV файл содержит:

-   общую аналитику
-   разбивку по категориям

## Архитектура

Проект построен по принципам слоистой архитектуры:

handlers → services → repositories → database

Слои:

-   handlers --- HTTP обработчики
-   services --- бизнес логика
-   repositories --- работа с базой данных
-   storage --- подключение к PostgreSQL
-   models --- структуры данных

## База данных

Используется PostgreSQL.

### Таблица items

  Поле         Тип
  ------------ -------------------------
  id           SERIAL PRIMARY KEY
  type         TEXT (income / expense)
  amount       NUMERIC(12,2)
  category     TEXT
  created_at   TIMESTAMP

Ограничения:

-   сумма не может быть отрицательной

## API

Базовый URL:

    http://localhost:8080/api

### Создать транзакцию

POST /items

``` json
{
  "type": "income",
  "amount": 1200,
  "category": "salary",
  "created_at": "2026-03-01T12:00:00Z"
}
```

### Получить список транзакций

GET /items

Фильтры:

    /items?type=income
    /items?category=food
    /items?from=2026-03-01T00:00:00Z&to=2026-03-31T23:59:59Z

### Обновить транзакцию

PUT /items/{id}

### Удалить транзакцию

DELETE /items/{id}

------------------------------------------------------------------------

## Аналитика

### Получить сводную аналитику

GET /analytics?from=...&to=...

### Группированная аналитика

GET /analytics/grouped?from=...&to=...&group_by=day

Варианты группировки:

-   day
-   week
-   category

------------------------------------------------------------------------

## Экспорт CSV

GET

    /analytics/export/csv?from=...&to=...

Скачивает файл с отчетом.
